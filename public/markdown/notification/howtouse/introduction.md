# Notification System Folder Structure and Instructions

## Folder Structure

The folder structure for implementing the notification 
system is as follows:

- **`/src`**
  - **`/components`**
    - **`/Notification`**
      - `NotificationStack.jsx`
      - `Notification.jsx`
      - `NotificationProvider.jsx`
      - `Types.jsx`
